Global ODP_SHM_SW_ONLY
When set, application will not share the reserved memory with HW accelerators. However, leaving this flag to zero does not guarantee that the reserved memory can be accessed from HW, and thus usage of this flag is considered deprecated. If HW accessible memory is required, set ODP_SHM_HW_ACCESS instead.
